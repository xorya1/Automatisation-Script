This is a script used for automating the following tasks for FAT disk types:
 Determine type of each partition that’s inside the partition table.
 Determine the starting offset for each partition’s VBR.
 Determine the number of partitions found.
 Determine whether the last partition in partition table is null or not.
 Determine the cluster size.
 Determine whether each partition is active or not by checking the byte number 0.
